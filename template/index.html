Service helps you to collect Open Graph layout of interested URL as JSON. Includes Open Graph, Facebook, Twitter, Mobile, VK tags. Just enter URL you need and press Collect Tags or Enter.

URL should begin from http:// or https://
Service API is available on /api/ Use following pattern to make GET request:
https://open-graph.gq/api/?url=URL_YOU_WANT_TO_PROCESS
